linux的基本用法

Share

Linux系统的优势：多用户、多任务的系统、安全性更高、开源免费，它将所有的目录和文件数据组织为一个树型的目录结构，整个系统中只存在一个根目录，所有的分区、目录、文件都在同一个根目录下面放

1、命令提示符

[root@localhost~]#

root:当前登录的用户名

localhost：本机的主机名

~：代表家目录

root 用户的家目录是：/root

普通用户的家目录是：/home

#：当前登录的用户为管理员用户

$：当前登录的用户为普通用户

2、切换用户用：su用户名

由 root 用户切换到普通用户不需要输入密码

由普通用户切换到其他用户就必须输入密码

3、显示当前目录所在的完整路径:pwd
4、目录切换命令：cd

cd /etc 切换到/etc/目录

cd .. 返回上一级目录

cd ~ 切换到home目录

cd - 切换到上一次访问的目录

根目录：/

家目录：普通用户的家目录在/home下面，root的家目录是/root

5、绝对路径和相对路径

相对路径：是当前目录下的某个文件或路径

绝对路径：是指目录或文件的完整路径

6、关机和重启

shutdown -h now 立即关机

shutdown -h 10 10分钟后关机

poweroff 立即关机

shutdown -r now 立即重启

shutdown -r 10 10分钟后重启

reboot 立即重启

7、显示目录内容

ls: 查看目录中的文件

ls -a: 显示隐藏文件（以.号开头的文件）

ls -l或ll: 显示详细文件

蓝色：目录文件 灰色：普通文件 绿色：可执行文件 红色：压缩文件 线绿色：（链接文件）快捷方式

8、通配符的使用

“？”可以匹配文件名中一个任意字符 ls -l /etc/v???

“*”可以匹配文件名中的任意多个字符

ls -l /etc/ns*.conf

9、创建、删除目录

mkdir mp3 mp4 mp5 创建3个子目录

mkdir -p /media/cdrom 创建连续目录

mkdir -m 775 tesst 创建目录，同时设置权限

mv mp3 mp4 移动/重命名一个目录

rm -r mp4 删除询问

rm -rf mp4 删除不询问

10、创建、删除文件

touch 1.txt

rm 1.txt 删除询问

rm -f 1.txt 删除不询问

11、移动和复制文件

cp 1.txt 2.txt 复制文件

cp -r dir1 dir2 复制目录

mv dir1 dir2 重命名目录

mv /mnt/dir1 /mnt 剪切目录

12、查看文件内容命令

cat：看最后一页，不适合查看长文件

more：百分比显示，回车、空格可以向下一页，q可以结束查看

less：翻页查看，键盘的上下键翻页，必须按q才能结束查看

head 10 file1：查看文件的前10行

tail 10 file1：查看文件的后10行

wc命令：文件内容的统计

语法：mc 选项 文件

-l：统计行数

-w：统计单词数

-c：统计字节数

13、查找命令
find命令：搜索文件与目录

作用：是在某个指定的路径下找我们需要的文件或目录，目标是文件或目录

语法：find [查找范围] [查找条件表达式]

-name：按名称查找，允许使用通配符

-type：按文件类型查找文件类型包括：普通文件：f 目录：d 块设备文件：b 字符设备文件：c

-user：按文件所有者查找，根据文件是否属于某个目标用户进行查找

-size：按文件大小查找，使用”+”、”-“号设置超过或小于指定的大小

find / -name file1 从”/“开始进入根文件系统查找文件或目录

find . -type d | sort 查找当前所有目录并排序

find / -user user1 查找属于用户”user1”的文件和目录

find . -size +10M 查找当前目录大于10M的文件

grep命令：查找文件内容

作用：是在某个文件中找我们所需要的某部分内容，目标是字符串

语法：grep [选项] 查找条件 目标文件

-c：计算找到”搜寻字符串”的次数

-i：忽略大小写的不同，所以大小写视为相同

-n：顺便输出行号

-v：反向选择

grep不支持通配符，而是使用正则表达式

^：表示以什么字符开头

$：表示以什么字符结尾

grep ^str /tmp/test 在文件”/tmp/test”中查找以”str”开始的

ps -ef | grep sshd 查找指定ssh服务进程

ps -ef | grep sshd | grep -v grep 查找指定服务进程，排除grep外

ps -ef |grep sshd -c 查找指定进程个数

14、其他辅助命令

查看历史命令：history

help命令：只能查看内部命令的帮助信息

man命令：查看命令的帮助手册，按q键退出

15、重定向、管道符”|”

输入重定向：<
输出重定向：>

“>>”:将命令结果重定向并追加到指定文件的末尾保存，而不覆盖文件中原有的内容

左边命令的执行结果作为 “|” 右边命令的输入

ls -l /etc/*.conf | wc –l

统计/etc 目录下所有以“.conf”结尾的文件的个数。

16、文件权限
文件属性：

1、文件类型，其中第一个字符代表文件的类别：
-：普通文件 d：目录 l：符号链接 c：字符设备 b：块设备
文件权限，r：可读 w：可写 x：可执行 -：没有权限
2、文件连接数
3、文件所有者
4、文件属组
5、文件大小，默认单位为字节
6、文件创建时间
7、文件名称

字母方式修改权限

文件所有则权限（u）
所属组权限（g）
其他用户权限（o）
所有用户（a）
举例：chmod u-w /mnt
chmod u+w /mnt

数字方式修改权限

r=4 w=2 x=1 -=0
chmod 777 test1

修改文件的所有者与所属组

chown :student /test //修改所属组
